# YOLO

YOLOv8 を用いた人物検出テストプロジェクトの一部です。
プライベートファイルを追加しないと動作しません。
ローカルカメラからリアルタイム推論を行います。

## 📦 使用技術
- Python 3.14.3
- YOLOv8（Ultralytics版）
- OpenCV
- PyTorch

## 🚀 実行方法

```bash
git clone https://github.com/rithub0/test_YOLO.git
cd test_YOLO
pip install -r requirements.txt
python main.py
